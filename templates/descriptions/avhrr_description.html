AVHRR

  • Description:The AVHRR Pathfinder Version 5.2 Sea Surface Temperature data set (PFV52) is a collection of global, twice-daily 4km sea surface temperature data.
  • Organization:NOAA
  • Spatial resolution: 4-km grid (1/24 deg)
  • Time Span: 1981-08-24 - 2012-12-31
  • Variables:
    • Sea Surface Temperature
